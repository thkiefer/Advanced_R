linmodEst {linmod}R Documentation

main estimation function

Description

Takes in a matrix x of predictors and computes linear model of y on x.

Usage

linmodEst(x, y)

linmod(x, ...)

Arguments

x

numeric perdictor matrix

y

numeric vector. Dependent variable.

...

further objects to be passed to methods

Value

List of 4 elements

  • coefficients coefficients of parameter estimation

  • vcov variance covaraiance matrix

  • sigma residual standard deviation

  • df degrees of freedom
